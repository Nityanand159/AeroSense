Welcome to AeroSense. This documentation provides developers and researchers with the tools needed to access high-quality air intelligence data.

Getting Started

To begin using our intelligence platform, you need to register for an API key in your dashboard. Once registered, you can start making requests to our global environmental monitors.

API Access

Use the following base URL for all API calls:

GET https://api.aerosense.io/v1/air-quality?lat=40.71&lon=-74.00

The response will return PM2.5, PM10, and NO2 levels in real-time.
