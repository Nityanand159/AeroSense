## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Basic knowledge of HTML and JavaScript

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Nityanand159/AeroSense.git
```

2. Navigate to the project directory:
```bash
cd AeroSense
```

3. Open the HTML file in your web browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

## 📖 Usage

Access the application through your browser at the appropriate localhost URL or open the HTML file directly.
